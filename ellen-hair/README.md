# Ellen Hair Institucional

Projeto Front-End institucional para a Ellen Hair — Skin BOULEVARD.

## Stack
- Next.js 14+ (App Router)
- React 19
- Tailwind CSS (v4)
- TypeScript
- Framer Motion

## Deploy Fácil (Vercel)
Este projeto está pronto para ambientes zero-config.
1. Suba o repositório para o GitHub.
2. Importe na Vercel.
3. Env variables: Nenhuma obrigatória para build estático puro.

## Setup Local
\`\`\`bash
npm install
npm run dev
\`\`\`
O site estará disponível na porta configurada (3000 padrão).

## Logos e Assets
Por favor, lembre-se de depositar os assets finais de fotografia e a logomarca do cliente (`logo.png`) diretamente no diretório `public/`.
